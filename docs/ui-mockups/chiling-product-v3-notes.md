# 赤灵AI运营工作台 v3 UI 设计说明

## 设计目标

- 面向运营人员，而不是工程人员：界面只展示“导入、设置、审核、生成、交付”等业务动作。
- 内部模型、供应商、接口和技术实现不出现在用户界面。
- 高风险动作必须有人工确认，尤其是肖像授权、素材授权、字幕规则和画面方向。
- 生成参数保持明确：单条 15 秒内、默认 1 条、单批最多 5 条、标准 480p / 高清 720p。

## 页面结构

1. **首页 / 任务队列**
   - 核心入口是“开始新作品”和“查看审核队列”。
   - 作品列表按状态展示：待审核、生成中、已交付。
   - 点击待审核作品后，右侧出现审核详情预览。

2. **创建作品 / 参数设置**
   - 三步流程：导入参考、设置生成、提交审核。
   - 支持粘贴视频链接、上传本地视频、上传人物/品牌授权素材。
   - 参数项可编辑：时长、清晰度、生成数量、字幕风格。

3. **人工审核 / 侧栏弹出**
   - 左侧保留参考视频预览，中间是文案和字幕规则编辑。
   - 右侧为审核详情侧栏，不离开主页面即可确认风险项。
   - 按钮分为“退回修改”和“确认并生成”，避免误操作。

4. **成品交付 / 下载归档**
   - 成品、字幕、审核记录分开下载。
   - 提供“下载交付包”和“复制分享路径”。
   - 发布前再次提醒确认人物授权和平台规范。

## 交互状态

- `待审核`：需要人工确认，进入审核侧栏。
- `生成中`：展示处理进度和预计时间。
- `已交付`：展示下载入口和归档记录。
- `确认并生成`：从审核页进入生产，只有审核项通过后可点击。
